Name: Effect
Summary: All effects will be able to follow this summary. 
         It contains a simple script that holds a effect type number

Description: 
    Interfaces- Effect type variable
    
    Integration- simply just gives a effect type variable which acts as an ID
                 to know which type of effect it is. You can simply drag this effect
                 to a weapon prefab to give this effect.

    Troubleshooting- Effect script allows you to adjust values of the effect to change
                     the strength or the frequency of effects being triggered.
